### 1. Audio-to-Text Transcript Chunker
* **The Problem:** Raw audio transcription results in long, unreadable walls of text that are difficult to align to video timelines or script formatting.
* **The Solution:** This tool uses the `SpeechRecognition` library to convert audio to text, then runs a custom list-parsing algorithm to chunk sentences into optimized, human-readable blocks saved straight to a text file.
* **Skills Shown:** Integrating external APIs, string tokenization, file I/O operations.

### 2. Spatial Layout & Scale Blueprint Calculator
* **The Problem:** Designing architectural or digital layouts requires continuous, tedious checking of area ratios, municipal open-space percentages, and functional walking clearances.
* **The Solution:** An engineering script that runs dimensional data through geometric and scaling algorithms to instantly output spatial configurations and flag clearance errors.
* **Skills Shown:** Algorithmic constraint handling, user input sanitization, structural control flow logic.

### 3. Number Guessing Game
* **What it does:** A classic CLI game where the user guesses a randomly generated number with dynamic feedback.
* **Concepts used:** `random` library, `while` loops, conditional logic, error handling.

### 4. Password Generator
* **What it does:** Instantly generates a strong, secure, randomized password based on user-defined length.
* **Concepts used:** `string` manipulation, list comprehensions, random sampling.

### 5. Command-Line To-Do List
* **What it does:** Allows users to add, view, and delete tasks in real-time through a terminal interface.
* **Concepts used:** Lists, `enumerate()`, user input routing.

### 6. Simple Command-Line Calculator
* **What it does:** Performs basic arithmetic (addition, subtraction, multiplication, division) based on user input.
* **Concepts used:** Functions, conditional execution, type casting (`float`), division-by-zero protection.

### 5. Rock, Paper, Scissors Game
* **What it does:** Lets the user play the classic game against an automated computer opponent.
* **Concepts used:** List lookups, `random.choice()`, nested evaluation logic.
